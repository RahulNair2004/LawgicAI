## Key Directories Explained:

### Frontend (`/frontend`)
- **components/**: Reusable React components
- **pages/**: Main page components with routing
- **services/**: API calls and external service integrations
- **utils/**: Helper functions, constants, and validation

### Backend (`/backend`)
- **models/**: MongoDB schema definitions and data models
- **routes/**: Flask route handlers (API endpoints)
- **services/**: Business logic (database operations, AI/RAG service)
- **config/**: Configuration files and database setup

### Data (`/data`)
- **legal_documents/**: Documents for RAG system training
- **sample_data/**: Seed data for development

## Environment Files Needed:

### Frontend `.env`:
```
REACT_APP_API_BASE_URL=http://localhost:5000
REACT_APP_ENVIRONMENT=development
```

### Backend `.env`:
```
MONGODB_URI=your_mongodb_atlas_connection_string
FLASK_ENV=development
SECRET_KEY=your_secret_key
AI_MODEL_API_KEY=your_ai_api_key
CORS_ORIGINS=http://localhost:3000
```
